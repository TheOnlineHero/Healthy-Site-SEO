== Description ==

A wordpress plugin that recommends which plugins to install to optimize and secure your site. It also recommends plugins/website links that can help boost your overall SEO score.

== Installation ==

1) Install WordPress 3.8 or higher

2) Download the latest from:

http://wordpress.org/extend/plugins/healthy-site-seo

3) Login to WordPress admin, click on Plugins / Add New / Upload, then upload the zip file you just downloaded.

4) Activate the plugin.
